Simple 3D Environment for Training Neural Networks

Built a minimal 3D simulation environment for training neural network agent. Implemented a custom ray tracing pipeline to generate spatial input data, combined with camera-based perception. Data is streamed via WebSockets to enable real-time interaction. The agent learns to navigate toward a target checkpoint.
